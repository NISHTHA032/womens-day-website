🌸 Women's Day Celebration Website

A simple and interactive **Women's Day celebration website** created using **HTML, CSS, and JavaScript**.
This project celebrates the strength, achievements, and inspiration of women around the world.

The website includes inspirational women, quotes, and a celebration button that triggers an animation with flowers and hearts.

 🌟 Features

* 🎉 Celebration button with animation
* 🌸 Falling flowers and hearts effect
* 👩 Inspirational women section
* 💬 Motivational quotes
* 🎨 Clean and responsive design
* 💻 Simple beginner-friendly code

🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript

🎉 How the Celebration Works

When the **Celebrate Women's Day** button is clicked:

* A **Happy Women's Day message appears**
* **Flowers and hearts fall from the top**
* Creates a small celebration animation
